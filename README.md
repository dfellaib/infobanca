Esercizio didattico per approcciare la OOP


## Configurazione Sublime 
Per aggiungere autosave, installare prima il package control (tool --> command palette --> Package Controll: install package) e successivamente il plugin di autosave

{
	"theme": "Default.sublime-theme",
	"font_size": 20,
	"ignored_packages":
	[
		"Vintage",
	],
	"keys": ["ctrl+shift+s"], "command": "auto_save"
}
